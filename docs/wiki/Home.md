# rapid7-mcp Wiki

Welcome to the `rapid7-mcp` wiki. This documentation is designed for security practitioners and builders who want to query InsightVM, InsightIDR, and Metasploit Pro with natural-language prompts.

## What you’ll find here

- **Tool catalogs** with practical use cases and copy/paste prompts
- **Role-based playbooks** (SOC, vulnerability management, pentest telemetry)
- **Prompt patterns** to get better results from your MCP client
- **Demo scenarios** you can run with `DEMO_MODE=true`
- **Troubleshooting** for common auth/query/result issues

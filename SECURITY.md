# Security Policy

Thanks for helping keep `rapid7-mcp` secure.

## Supported Versions

This project currently supports security fixes on the latest `main` branch.

## Reporting a Vulnerability

Please **do not** open public GitHub issues for suspected vulnerabilities.

Instead, report privately by email:

- **Email:** `security@securitah.io`
- **Subject:** `rapid7-mcp security report`

Please include:

- A clear description of the issue
- Steps to reproduce
- Potential impact
- Any suggested remediation

## Response Expectations

- Initial acknowledgment: within **3 business days**
- Triage/update: within **7 business days**
- Fix timeline: depends on severity and complexity

We appreciate responsible disclosure and will credit reporters (with permission).
